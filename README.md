# 🚀 Productivity Dashboard

A modern and interactive **Productivity Dashboard** built using **HTML, CSS, and JavaScript** to help manage daily tasks, goals, and focus sessions efficiently.

---

## ✨ Features

- ✅ To-Do List (Add, delete, mark important tasks)
- 📅 Daily Planner (Hourly scheduling system)
- 💡 Motivation Quotes (API-based random quotes)
- ⏳ Pomodoro Timer (Work & break sessions)
- 🎯 Daily Goals Tracker (with completion status)
- 🌦️ Live Weather + Date & Time
- 🌗 Light / Dark Mode Toggle (High contrast UI)

---

## 🎨 UI Highlights

- Clean and modern design
- Fully responsive layout
- Smooth animations and transitions
- **Enhanced Dark Mode with pure white text for better readability**

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Custom Properties / Variables)
- JavaScript (Vanilla JS)
- LocalStorage (Data persistence)
- Weather API & Quotes API
